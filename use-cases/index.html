Automata — distributed edge-to-cloud platform

Designed and delivered a distributed edge-to-cloud platform architecture, owning end-to-end platform design, security model, and operational standards across AWS and edge environments.

  • Architected a centralised cloud control plane on AWS EKS, acting as the authoritative management layer for global edge infrastructure.
  • Designed and implemented a custom control panel using Python and Temporal to orchestrate long-running workflows for provisioning, lifecycle management, and remediation of edge clusters.
  • Built a fully automated infrastructure-as-code platform using Terraform, defining standardised, repeatable environments and enforcing platform consistency across regions.
  • Designed an edge cluster provisioning system based on Talos Linux, enabling secure, immutable, and remotely managed Kubernetes clusters at the edge.
  • Integrated Argo CD for GitOps-based configuration and application delivery, establishing declarative, auditable, and repeatable cluster state management across cloud and edge.
  • Defined platform-wide security architecture, including identity, access control, secrets management, cluster isolation, and secure bootstrap of edge environments.
  • Designed networking architecture spanning cloud and edge, including cluster connectivity, segmentation, ingress/egress patterns, and secure control plane communication.
  • Established operational standards for cluster lifecycle (provisioning, upgrades, recovery, and decommissioning), enabling scalable management of large fleets of Kubernetes clusters.
  • Acted as both hands-on platform engineer and solution architect, translating high-level business and operational requirements into a robust, production-grade distributed systems architecture.

Key technologies: AWS, EKS, Kubernetes, Talos Linux, Terraform, Python, Temporal, Argo CD, GitOps, cloud & edge architecture, platform security, distributed systems, infrastructure architecture.
